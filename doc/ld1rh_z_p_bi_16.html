LD1RH: Load and broadcast unsigned halfword to vector

128-bit SVE

Load a single 16-bit value from the memory operand (1), and broadcast it to all 16-bit elements of (2). If all predicate bits corresponding to elements in (2) are zero, the load is skipped, and cannot cause a fault.
