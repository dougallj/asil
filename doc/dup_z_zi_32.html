128-bit SVE

Set all 32-bit lanes of (2) to the element from (1) specified by imm (or zero, if imm ≥ 4).

256-bit SVE

Set all 32-bit lanes of (2) to the element from (1) specified by imm (or zero, if imm ≥ 8).

512-bit SVE

Set all 32-bit lanes of (2) to the element from (1) specified by imm.
Larger sizes

1024-bit SVE

Set all 32-bit lanes of (2) to the element from (1) specified by imm. The immediate is limited to 0 ≤ imm < 16, so only elements from the low half of (1) may be specified on 1024-bit SVE.

2048-bit SVE

Set all 32-bit lanes of (2) to the element from (1) specified by imm. The immediate is limited to 0 ≤ imm < 16, so only elements from the low quarter of (1) may be specified on 2048-bit SVE.
